### [valueOf 和 toString](https://juejin.cn/post/6844903967097356302)

| 对象     | valueOf              | toString                                                     |
| -------- | -------------------- | ------------------------------------------------------------ |
| Array    | 数组本身             | 以逗号分隔的字符串                                           |
| Boolean  | 布尔值               | “布尔值”                                                     |
| Date     | 返回毫秒形式的时间戳 | 可读的时间字符串，如“Mon Mar 27 2023 19:36:43 GMT+0800 (中国标准时间)” |
| Function | 函数本身             | 声明函数的JS源代码字符串                                     |
| Number   | 数字值               | “数字值”                                                     |
| Object   | 对象本身             | “[object Object]”                                            |
| String   | 字符串值             | “字符串”                                                     |
